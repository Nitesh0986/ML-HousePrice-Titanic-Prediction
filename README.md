# ML-HousePrice-Titanic-Prediction
Machine Learning project demonstrating Linear Regression for House Price Prediction and Logistic Regression for Titanic Survival Prediction using Python, Pandas, Scikit-Learn, and Matplotlib
# ML House Price & Titanic Prediction

## Overview

This repository contains two Machine Learning projects built using Python and Scikit-Learn.

### Project 1: House Price Prediction

A Linear Regression model that predicts house prices based on house area.

### Project 2: Titanic Survival Prediction

A Logistic Regression model that predicts passenger survival using features such as age, gender, and passenger class.
